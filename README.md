# 🌟 Project Name
🛒 A-6 Digital Tools Buying Website
## 📌 Description

A modern and responsive digital tools buying website where users can explore premium tools, view product details, and easily add items to their cart. The platform provides a smooth and user-friendly shopping experience with dynamic cart management and interactive design.

---

## 🚀 Technologies Used

* ⚛️ React
* 🎨 Tailwind CSS
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎯 CSS3

---

## ✨ Features

✨ Features
🛒 Dynamic Cart System
Users can add and remove products from the cart, and the cart count updates instantly in the navbar.
🔄 Product & Cart Toggle System
Easily switch between product view and cart view with interactive buttons.
🔔 Toast Notifications
Integrated React-Toastify to show alerts for add to cart, remove product, and checkout actions.
